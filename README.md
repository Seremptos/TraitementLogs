# TraitementLogs

Cette application sert à traiter des logs et à les afficher.

Elle est composée de plusieures parties :

- Un reverse proxy (nginx)
- Un frontend (Vue.js)
- Un backend (Flask)
- Une API pour communiquer avec la base de données (ASP.NET Core)
- Une base de données relationelle (Postgres)

Cette application est déployée à l'aide de Docker qui est une technologie de conteneurisation d'applications open-source.
